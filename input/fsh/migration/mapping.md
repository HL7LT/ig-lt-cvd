# CVD attachments vs ig-lt-cvd examples тАФ mapping

**Questionnaire тЖФ profile mapping:** see the IG page **Questionnaire mapping** (`questionnaire-mapping.md` / `questionnaire-mapping.html`) and the ConceptMaps `cvd-risk-assessment-questionnaire` and `cvd-prevention-plan-questionnaire`.

Comparison of **attachment expectations** (Word: CVD questionnaire + prevention plan + achievement; Excel: Core Entities, Clinical Measurement Details, Relationships) to **examples in ig-lt-cvd**, with analogues in **ig-lt-base**, **ig-lt-vitalsigns**, **ig-lt-lifestyle**, **ig-lt-lab**.

## Summary

**Present in ig-lt-cvd examples:** Patient, Practitioner, Organization, CVDRiskAssessmentLtCvd, RiskGroupObservationLtCvd, several RiskFactorStatusLtCvd, one CvdChronicConditionLtCvd, EKGLtCvd, CarePlanLtCvd + Goals (LDL/BMI/BP), generic MedicationStatement, ObservationLt diabetes duration, duplicate care-plan variants (inline/external).

**Missing from ig-lt-cvd:** Encounter, PractitionerRole, almost all vitals and labs used in the questionnaire, lifestyle observations (smoking, alcohol, activity, diet), family CVD history, referral (ServiceRequest), follow-up appointment (Appointment), creatinine/eGFR, and separate тАЬachievementтАЭ round examples (same types as vitals/labs/lifestyle at a second timepoint).
